Downloading Dontprint version 0.2beta2...

You should see a warning message about installing new software in the upper part of this window. Click on "Allow" to install Dontprint.
Dontprint is an add-on for the Mozilla Firefox web browser and the installation message is only displayed if you're using Firefox. You can download Firefox for free at firefox.com.

This is a beta version

Please be aware that you are installing a beta version of Dontprint. The main features work already but it is still a bit rough around the edges and some features aren't even implemented yet. If you experience any problems using Dontprint please file a bug report.

Can't see any installation warning?

At the moment Dontprint requires the Mozilla Firefox web browser. If you don't see any installation warning in the upper part of this window then you are probably using a different web browser. Download Firefox for free from firefox.com.

Note: Some other add-ons make Firefox identify itself as a different browser. If you believe that you are in fact using Firefox then click the button below.
